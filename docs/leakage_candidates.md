# 누수 후보 분류표 — 32개 컬럼

> **이 문서의 목적:** 32개 컬럼 각각에 대해 *예약이 접수되는 시점(t=예약일)에 값이 확정되는가*를 판정해, 모델 입력으로 안전한 컬럼 / 위험한 컬럼 / 회의로 결정할 컬럼을 분류한다.
>
> **판정 원칙:** 단일 질문 — **"손님이 예약 버튼을 누른 그 순간에, 이 값이 이미 확정되어 있는가?"**
