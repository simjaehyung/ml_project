## 1. PM 가이드 결과

| 연도 | n | `previous_cancellations≥1` 비율 |

→ 2015 → 2016 사이 ≥1 비율이 20% → 3%로 급감. design_04 섹션 2-3에서 PM이 언급한 "감소 추세 비직관적"과 일치.

### 1-2. ≥1 그룹 취소율 (기대값 90%+)

| 연도 | n | 취소율 | 평가 |
|---|---:|---:|---|
| 2015 | 4,435 | **98.96%** | ✓ 기대값 충족 |

→ 2016년 단독으로는 신호 강도 약화. 다만 그룹 특성([E3])은 매우 강한 패턴이라 PR-AUC 기여도는 SHAP에서 별도 확인 필요.

## 2. ★ 미결 항목 C — 정의 어긋남 분석 (핵심 발견)

`is_repeated_guest=0` 인데 (= 재방문 손님 아닌데) `previous_cancellations≥1` 인 (= 과거 취소 이력 있는) 행을 분석.

### 2-1. 그룹 특성

| 항목 | 값 |
|---|---:|
| 행 수 | **5,520건** (전체의 7.01%) |
| 취소율 | **99.15%** |
| `lead_time` 평균 | **217일** (전체 평균 101일의 ~2.15배) |
| `lead_time` 중앙값 | 257일 |
| `market_segment` Groups + Offline TA/TO | **85.2%** (Groups 56.6% + Offline TA/TO 28.6%) |
| 호텔 분포 | City Hotel 84% (취소율 99.98%) / Resort Hotel 16% (94.73%) |
| `previous_cancellations` 값 분포 | 값 1이 5,336건 (97%) |

### 2-2. ★ `deposit_type=Non Refund`와 패턴 일치

design_04 섹션 2-6에서 PM이 발견한 `deposit_type=Non Refund` 그룹과 비교:

| 특성 | `deposit_type=Non Refund` | E3 정의 어긋남 그룹 |
|---|---|---|
| 취소율 | 99.2% | **99.15%** |
| Groups + Offline TA/TO | 61% + 37% = 98% | 56.6% + 28.6% = **85.2%** |
| `lead_time` 배수 (전체 평균 대비) | 2.2배 | **2.15배** |
| 행 수 | 10,461건 | 5,520건 |

→ **같은 B2B 단체 블록 패턴이거나 강하게 겹치는 그룹**으로 판단. PM의 가설 A (allotment 블록 반납 = 실제 재정 손실 없는 취소)와 정합.

### 2-3. 비교군: 정의 정상 그룹 (`is_repeated_guest=1 & ≥1`)

| 항목 | 값 |
|---|---:|
| 행 수 | 686건 (전체의 0.87%) |
| 취소율 | **61.37%** (덜 극단적) |
| `lead_time` 평균 | 104일 (전체 평균과 비슷) |

→ 진짜 재방문 손님이 과거 취소한 경우는 별개 그룹. 신호도 약함 (61% vs 99%).

### 2-4. 교차표 — `previous_cancellations≥1` 신호의 정체

|  | prev=0 | prev≥1 | 합계 |
|---|---:|---:|---:|
| `is_repeated_guest=0` | 70,764 | **5,520** ← B2B 블록 | 76,284 |
| `is_repeated_guest=1` | 1,733 | 686 ← 정의 정상 | 2,419 |
| 합계 | 72,497 | 6,206 | 78,703 |

→ `previous_cancellations≥1` 신호의 **89%(5,520/6,206)** 가 사실 *재방문 손님 과거 취소* 가 아니라 *B2B 블록 패턴*.

## 3. 결론 / 권고

### 3-1. 결정

- **18번 `previous_cancellations` KEEP 유지** (PM 결정 그대로). 신호 자체는 매우 강력 (≥1 그룹 취소율 94.97%). 단, *컬럼명이 의미하는 정의 ("같은 손님의 과거 취소 이력") 와 실제 신호 정체 (B2B 블록 패턴) 가 다르다*는 점을 인지하고 진행.

### 3-2. Week 3 SHAP 추가 감시 항목 ⚠

design_04의 미결 항목 A (`deposit_type` SHAP 1~2위 독식 여부)에 다음을 추가로 모니터링:

- **`previous_cancellations` 도 같은 B2B 블록 패턴을 잡는 중복 신호일 가능성 높음.**
- SHAP에서 두 변수가 *같은 행에 동시에 큰 기여*를 보이는지 확인.
- 만약 동시 기여 패턴이 발견되면 → 두 변수가 같은 의미를 다른 각도로 잡는 것 → 모델 해석 시 주의.

### 3-3. Phase 2 (Week 5) ablation 추가

design_04 미결 항목 E (`previous_cancellations` 제거 ablation)에 다음을 추가:

- `deposit_type` 와 `previous_cancellations` 를 *각자* 제거 vs *둘 다* 제거 시 PR-AUC 변화 비교.
- 두 변수가 중복 신호라면, 한쪽만 제거 시 손실은 작고 둘 다 제거 시 손실이 큼 → 통합 신호 가설 입증.

### 3-4. 2016년 ≥1 그룹 취소율 84.98% 약화

- 전체 평균(94.97%)보다 약 10%p 낮음. 2015 데이터의 영향이 평균을 끌어올린 것.
- test set (2017-03 ~ 2017-08)에서 어떻게 작동할지 Week 3 모델 평가 + SHAP에서 별도 확인.
